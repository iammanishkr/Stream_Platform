BeatStream - Your Music Companion 🎵 🎶

BeatStream is a sleek, web-based music streaming app built with Django & TailwindCSS. Explore, play, and vibe to your favorite songs with a dynamic music player and a responsive UI.

Features ✨
- User Auth (Signup/Login) 🔒
- Interactive Music Player (Play/Pause/Seek) 🎧
- Song Library with cover images, titles, artists 📚
- Artist Collection with dynamic cards 🖼️
- Mobile Responsive UI 📱

Tech Stack 💻
Frontend: HTML, TailwindCSS, JavaScript
Backend: Django, Python
Database: SQLite

Getting Started 🚀
1. Clone: git clone https://github.com/iammanishkr/Stream_Platform.git
2. Install: pip install -r requirements.txt
3. Migrate: python manage.py migrate
4. Run: python manage.py runserver
5. Visit: http://127.0.0.1:8000/
